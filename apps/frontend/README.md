# Frontend Admin - KimHoan

## Environment

Tạo file `.env`:

```env
VITE_API_BASE_URL=http://localhost:8000/api/v1
```

## Run

```bash
npm install
npm run dev
```

## Modules

- Đăng nhập (`/login`)
- Dashboard (`/`)
- Hàng hóa (`/products`)
- Khách hàng (`/customers`)
- Đơn hàng (`/orders`)
- Báo cáo (`/reports`)
